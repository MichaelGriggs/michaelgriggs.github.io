"Ac-Climb-atize" Color Scheme

'acclimbatize.com'

  • Home
  • Target Audience
  • Color Scheme
  • Typography

The headers and footers should all be the same color of mid-to-light-blue (VSCode calls it 'steelblue'). The elements the user will need to interact with should be a darker color with lighter text for accessibility. The background of the site should be climbing-related images at partial opacity to give them a faded effect. The company's logo should be a brighter color that catches attention.
